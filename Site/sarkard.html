À propos du projet

Architecture, fonctionnalités et apprentissages

Concept

SarKard est une plateforme de cartes à collectionner en ligne. Les joueurs s'inscrivent, ouvrent des boosters pour obtenir des cartes de différentes raretés (commune, rare, légendaire), et peuvent les revendre librement sur le marché à d'autres joueurs. Le prix est fixé par le vendeur, créant une véritable économie de marché virtuelle.

Fonctionnalités

  • Inscription / Connexion sécurisée
  • Ouverture de boosters avec drop aléatoire
  • Marché libre : achat et vente entre joueurs
  • Filtrage par rareté sur le marketplace
  • Collection personnelle avec gestion de vente
  • Système de crédits et profil utilisateur
  • Boutique pour acheter des boosters

Architecture technique

Application web MVC classique avec séparation front/back et base de données relationnelle.

Front-end HTML / CSS / JS
Back-end PHP 8 / Sessions
Base de données MySQL / PDO
Serveur XAMPP / Apache

Ce que j'ai appris

  • Développement full-stack PHP/MySQL
  • Modélisation de BDD relationnelle
  • Opérations CRUD complètes
  • Gestion des sessions et authentification
  • Requêtes SQL préparées (sécurité)
  • Logique métier complexe (transactions)
